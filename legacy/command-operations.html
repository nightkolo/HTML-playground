File Operations

Action Command Example
Rename a file mv old_name new_name mv file.txt newfile.txt
Duplicate a file cp filename duplicate_name cp file.txt file_copy.txt
Move a file mv filename /path/to/destination/ mv file.txt ~/Documents/
Delete a file rm filename rm file.txt
Open a file open filename open file.txt
Get file info ls -l filename ls -l file.txt

Directory Operations

Action Command Example
Create a directory mkdir directory_name mkdir new_folder
Copy a directory cp -r source_dir destination_dir cp -r folder1 folder2
Move a directory mv source_dir /path/to/destination/ mv folder1 ~/Documents/
Delete a directory rm -r directory_name rm -r folder1
List contents ls ls ~/Documents
Show hidden files ls -a ls -a ~/Documents

Search & Compression

Action Command Example
Search for a file find /path -name "filename" find ~/Documents -name "*.txt"
Compress files/dir zip -r archive_name.zip file_or_dir zip -r archive.zip folder1
Extract a ZIP file unzip archive_name.zip unzip archive.zip

Quick Tips

  • Replace /path/to/destination/ with the target folder's path.
  • Use ~ as a shortcut for the home directory.
  • Hidden files start with . (e.g., .hiddenfile), and you can view them with ls -a.
  • Be cautious with rm: it permanently deletes files or folders.
